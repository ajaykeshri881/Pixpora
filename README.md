# Pixpora

Pixpora is a modern media search web application that allows users to discover **images and videos** from multiple popular platforms in one place.

It integrates the **Unsplash** and **Pexels** APIs to deliver high-quality visual content with a smooth and interactive user experience.

[![Go to Website](https://img.shields.io/badge/Go%20to%20Website-Visit-blue?style=for-the-badge)](https://pixpora.pages.dev/)


---

## Features

- Search images from Unsplash  
- Search videos from Pexels  
- Fast and responsive UI  
- Smooth animations with Framer Motion  
- Global state management using Redux  
- Toast notifications for better UX  
- Progressive Web App (PWA) support  

---

## Tech Stack

- React (Vite)
- Redux Toolkit
- React Router
- Framer Motion
- Tailwind CSS
- Unsplash API
- Pexels API
