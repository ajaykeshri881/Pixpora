# Pixpora

Pixpora is a modern media search web application that allows users to discover **images, videos, and GIFs** from multiple popular platforms in one place.

It integrates the **Unsplash**, **Pexels**, and **Giphy** APIs to deliver high-quality visual content with a smooth and interactive user experience.

---

## Features

- Search images from Unsplash  
- Search videos from Pexels  
- Search GIFs from Giphy  
- Fast and responsive UI  
- Smooth animations with Framer Motion  
- Global state management using Redux  
- Toast notifications for better UX  
- Progressive Web App (PWA) support  

---

## Tech Stack

- React (Vite)
- Redux Toolkit
- React Router
- Framer Motion
- Tailwind CSS
- Unsplash API
- Pexels API
- Giphy API
